## How to Capture Screenshots

1. Open the published dashboard
2. Navigate to each page
3. Use browser's full-page screenshot (or Cmd+Shift+4 on Mac / Win+Shift+S on Windows)
4. Save with the filenames above
5. Recommended: 1920×1080 resolution, PNG format
6. Upload via GitHub: Code → images/ → Add file → Upload files
